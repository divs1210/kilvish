    • Kilvish

    A sprite-based 2D Game Engine written purely on top of Java AWT/Swing.

    Project maintained by divs1210 Hosted on GitHub Pages — Theme by mattgraham

    Why bother?

    • Making games is fun and rewarding.
    • Kilvish is simple and intuitive to use, and a fun way to learn programming.

    Try it out

    • Import the eclipse project into your IDE.
    • Check out the examples.
    • Copy the res/media folder into your bin/ folder if you get FileNotFound errors.
